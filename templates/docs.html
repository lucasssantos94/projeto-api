Documentação da API

Esta é a documentação da API para gerenciamento de livros.

1. Listar Livros

Descrição: Retorna uma lista de todos os livros cadastrados.

Método: GET

URL: /livros

Exemplo de Resposta:

{
    "status": "success",
    "data": [
        {
            "id": 1,
            "titulo": "Dom Casmurro",
            "categoria": "Literatura Brasileira",
            "autor": "Machado de Assis",
            "imagem_url": "https://exemplo.com/dom_casmurro.jpg"
        }
    ]
}

2. Doar Livro

Descrição: Adiciona um novo livro ao banco de dados.

Método: POST

URL: /doar

Corpo da Requisição (JSON):

{
    "titulo": "Novo Livro",
    "categoria": "Ficção",
    "autor": "Autor Teste",
    "imagem_url": "https://exemplo.com/imagem.jpg"
}

Exemplo de Resposta:

{
    "status": "success",
    "message": "Livro doado com sucesso!"
}

3. Editar Livro

Descrição: Edita as informações de um livro existente.

Método: PUT

URL: /editar/<int:livro_id>

Corpo da Requisição (JSON):

{
    "titulo": "Livro Atualizado",
    "categoria": "Não Ficção",
    "autor": "Autor Atualizado",
    "imagem_url": "https://exemplo.com/nova_imagem.jpg"
}

Exemplo de Resposta:

{
    "status": "success",
    "message": "Livro editado com sucesso!"
}

4. Deletar Livro

Descrição: Remove um livro do banco de dados.

Método: DELETE

URL: /deletar/<int:livro_id>

Exemplo de Resposta:

{
    "status": "success",
    "message": "Livro deletado com sucesso!"
}
